# SolarMatrix 🪐

SolarMatrix is a structured web application designed to visualize and compare planetary data through a clean, data-first interface. This project serves as a comprehensive exploration of our solar system, utilizing semantic HTML markup to organize complex scientific information into an accessible digital experience.

# 🚀 Overview

The goal of SolarMatrix is to bridge the gap between raw astronomical data and visual storytelling. It features a hierarchical layout that moves from high-level exploration to granular, technical comparisons, making it an ideal tool for both educational purposes and data visualization practice.

# ✨ Key Features

## Data-Driven Tables: 
A multi-layered comparative matrix detailing planetary mass, diameter, gravity, and orbital velocity.

## Multimedia Integration: 
Embedded video and image placeholders designed for high-impact visual context.

## Structured Markup: 
Built with a focus on clean, semantic HTML5 for optimal accessibility and information hierarchy.

## Interactive Inquiry: 
Includes a comprehensive "Planetary Science" contact and inquiry form with multi-select options for user engagement.

## Responsive Design: 
A vertical-flow layout optimized for clarity across different screen dimensions.

# 🛠️ Tech Stack

1. HTML5: Semantic structure and advanced table layouts.

2. CSS3: Modular styling for data visualization and form elements.

3. Information Design: Focused on hierarchy, readability, and data density.

## Getting Started

To get a local copy and running follow the following steps.

1. Clone the app by running this command

    git clone ["Click here"](https://github.com/JacintaNgwu/SolarMatrix.git)

2. Navigate into the folder by running this command

    cd My-Portfolio

3. Install the dependencies by running this command

    npm install

4. View the output in your browser by running this command

    npm start
